# Eternal Invasion: Drone Edition

`drone/index.html` 하나로 실행되는 Canvas 2D 자동 방어 게임입니다. 4x4 생산 그리드에서 드론이 자동으로 생성, 이동, 병합되고, 완성된 드론이 중앙 코어 주변으로 출격해 외곽에서 몰려오는 적을 요격합니다.

## 실행 방법

브라우저에서 `drone/index.html` 파일을 열면 바로 시작됩니다. 별도 빌드나 서버는 필요 없습니다.

## 현재 게임 구조

- HTML, CSS, JavaScript가 모두 `index.html`에 들어 있는 단일 파일 구조입니다.
- 렌더링은 고정 크기 `1000x600` Canvas 2D를 사용하고, HUD는 CSS 오버레이로 표시합니다.
- 주요 클래스는 `Game`, `Drone`, `Projectile`, `Particle`입니다.
- `TECH_TREE`가 레벨별 드론 능력치와 Assault/Defense 분기를 정의합니다.
- Web Audio API로 발사, 피격, 병합, 코어 충격음을 합성하며 기본값은 음소거입니다.

## 핵심 플레이 흐름

- 시작 후 5초 동안 준비 시간이 주어집니다.
- 매 1초마다 한 턴이 진행됩니다.
- 자동 루틴은 빈 칸에 Scout를 생산하고, 같은 레벨/경로의 인접 드론을 병합하며, 위협 상황에서는 고레벨 드론을 출격시킵니다.
- 플레이어는 그리드 칸을 클릭하거나 터치해 특정 드론을 출격 대기열에 넣을 수 있습니다.
- 이미 대기열에 있는 칸을 다시 누르면 출격 예약이 취소됩니다.
- 코어 HP가 0이 되면 결과 화면을 보여주고 7초 뒤 소프트 리셋됩니다.

## 개선 작업 요약

- 프레임레이트 의존 로직을 고정 타임스텝 루프로 변경했습니다.
- `Date.now()` 기반 준비 시간과 턴 처리를 60Hz 시뮬레이션 틱으로 통일했습니다.
- 코어 접촉 시 매 프레임 사운드와 화면 흔들림이 반복되던 문제를 쿨다운 방식으로 완화했습니다.
- `location.reload()` 재시작을 게임 상태만 초기화하는 소프트 리셋으로 교체했습니다.
- 점수, 웨이브, 격파 수, 병합 수, 최고 점수를 추가했습니다.
- 정적 설명 패널을 실시간 이벤트 로그로 교체했습니다.
- 모바일 터치를 지원하도록 `pointer` 입력을 적용했습니다.
- 작은 화면에서 HUD가 Canvas를 과도하게 가리지 않도록 반응형 CSS를 추가했습니다.
- 엔티티 수 상한을 두어 후반부 성능 저하 위험을 줄였습니다.
- 깨진 README 문서를 정상 UTF-8 한글 문서로 복구했습니다.

## 검증 포인트

- 60Hz와 고주사율 화면에서 드론 이동, 턴, 스폰 속도가 크게 달라지지 않아야 합니다.
- 게임 시작 직후 5초 준비 카운트다운이 정상 표시되어야 합니다.
- 그리드 칸 클릭/터치로 출격 예약과 취소가 모두 동작해야 합니다.
- 코어가 공격받을 때 사운드와 흔들림이 과도하게 중첩되지 않아야 합니다.
- 게임 오버 후 점수 요약과 최고 점수가 표시되고, 7초 뒤 자동으로 새 게임이 시작되어야 합니다.
- 모바일 폭에서 HUD와 버튼 텍스트가 서로 겹치지 않아야 합니다.

## 남은 개선 후보

- 드론 병합 경로 선택을 무작위가 아니라 현재 위협 상황에 맞게 조정하기
- 웨이브별 적 조합을 테이블화해 난이도 곡선을 더 명확히 만들기
- Canvas 내부에 웨이브 배너와 코어 경고 연출 추가하기
- `index.html`이 더 커지면 CSS/JS를 파일로 분리하기
