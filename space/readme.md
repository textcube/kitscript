# Eternal Invasion: Space Edition (Space Auto-Battle)

캔버스 기반 오토배틀 스페이스 인베이더. 단일 `index.html` 파일(외부 의존성 없음, 바닐라 JS + Canvas 2D + WebAudio)로 구성되어 있다.

## 현재 버전 개요

### 핵심 시스템
- **오토배틀**: 플레이어 기체가 자동으로 목표를 고르고(`pickTarget` — HP가 가장 낮은 적 우선, 동률이면 가까운 적) 목표를 향해 이동·조준 사격한다. 조준선(점선)이 목표까지 표시된다.
- **수동 오버라이드**: ←/→ 또는 A/D 키 입력 중에는 수동 이동이 자동 이동보다 우선한다.
- **웨이브 시스템**: 6×3 적 편대. 줄마다 기본 HP 1/2/3, 3웨이브마다 전체 HP +1. 웨이브가 오를수록 편대 이동 속도(`+8/wave`)와 하강 폭이 증가. 전멸 시 다음 웨이브 즉시 시작(총알 초기화).
- **편대 이동**: 고전 인베이더 방식 — 좌우 왕복하다 벽에 닿으면 반전(`vx *= -1.06`으로 점점 가속)하며 한 칸 하강.
- **적 공격**: 0.7~1.2초 간격으로 생존 적 중 무작위 1기가 플레이어 방향(±12px 오차)으로 조준탄 발사. 탄속은 `240 + wave*8`.
- **패배 조건**: 목숨 3개 소진, 또는 적 편대가 위험선(화면 하단 120px, 빨간 선)에 도달.
- **피격 무적**: 피격 후 0.8초 무적 + 기체 점멸 + 실드 이펙트.
- **게임 오버 → 자동 재시작**: 오버레이 표시 후 7초 카운트다운, 즉시 재시작 버튼 제공.

### 연출 / UI
- 절차적 렌더링 우주선(그라디언트 기체, 엔진 글로우), 적기(피격 시 플래시 컬러 스왑), 적별 HP 바.
- 시차(z값) + 반짝임이 있는 스타필드 130개.
- WebAudio 신시사이저 SFX 5종(발사/피격/격추 등), 상단 바 Sound ON/OFF 토글, 첫 입력 시 오디오 언락.
- HUD: 점수 / 목숨 / 웨이브 / 현재 목표 HP.
- 상단 바: 게임 타이틀, BuntGames 스튜디오명, YouTube·Kitscript 링크.

## 알려진 버그

1. **편대 반전 무한 루프 가능성** — 벽 반전 시 `vx *= -1.06`으로 속도만 뒤집고 위치를 보정하지 않아, 창이 좁거나 속도가 커지면 다음 프레임에도 경계 밖에 있어 매 프레임 반전+하강이 반복될 수 있다(편대가 급강하하며 즉사).
2. **포커스 이탈 시 키 고착** — 키를 누른 채 창 포커스를 잃으면 `keyup`을 못 받아 해당 방향으로 계속 이동한다(`blur` 시 keys 초기화 필요).
3. **HiDPI 흐림** — 캔버스 크기를 CSS 픽셀로만 잡아 레티나/고배율 디스플레이에서 렌더링이 흐릿하다(`devicePixelRatio` 미적용).
4. **모바일 미지원** — 키보드 입력만 받으므로 터치 기기에서는 자동 조작만 가능하고 수동 개입이 불가능하다.
